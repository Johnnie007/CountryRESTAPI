Country Rest API

Welcome to the Country RESTful API Created by: Johnnie Hicks

This API is very simple to use with a few ENDPOINT to retrieve data

Get all the Countries

Example

http://localhost:7000/world/

{ "id": "1", "name": "Afghanistan", "language":"Dari", "hemisphere":"North East", "continent":"Asia", "currency":"AFN", "population": "38,928,346", "capital": "Kabul" }, { "id": "2", "name": "Albania", "language":"Albanian", "hemisphere":"North East", "continent":"Europe", "currency":"ALL", "population": "2,877,797", "capital": "Tirana" }, { "id": "3", "name": "Algeria", "language":"Arabic", "hemisphere":"North East", "continent":"Africa", "currency":"DZD", "population": "43,851,044", "capital": "Algiers" }

Get Country by Name

Example

http://localhost:7000/world/USA

Recieve an array of objects by the Countries location(hemisphere). The hemisphere endpoints are NorthWest, SouthWest, NorthEast, and SouthEast

Example

http://localhost:7000/world/NorthEast

[ { "id": "5", "name": "Angola", "language": "Portuguese", "hemisphere": "South East", "continent": "Africa", "currency": "AOA", "population": "33,866,272", "capital": "Luanda" }, { "id": "9", "name": "Australia", "language": "English", "hemisphere": "South East", "continent": "Australia", "currency": "AUD", "population": "25,499,884", "capital": "Canberra" }, { "id": "23", "name": "Botswana", "language": "English", "hemisphere": "South East", "continent": "Africa", "currency": "BWP", "population": "2,351,627", "capital": "Gaborone" } ]
